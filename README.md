ExCrypt
=======
Encrypts folders and files with AES.

##Requirements for script:

- Visual C++ 2010 (manual installation)
- Python 3


###Install AES:

`pip install pycrypto` or use a [windows installer](https://github.com/axper/python3-pycrypto-windows-installer).
